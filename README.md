## Usage

> Webpack compile: 'npm run react-dev'
```sh
webpack -d --watch
```
> Start Server: 'npm run start'
```sh
nodemon server/index.js
```
> Seed DB: 'npm run db:setup'
```sh
node database/seed.js
```
